Resumo
O daltonismo é uma deficiência que atinge cerca de 8% da polução masculina mundia. Raros são os casos de daltonismo em que as pessoas enxergam apenas tons de cinza, a maioria dos daltônicos enxergam as cores mas possuem dificuldade em distingui-las. Essa dificuldade de diferenciar as cores cause problemas em atividades cotidianas de pessoas portadoras de daltonismo. Com a finalidade de minimizar esses problemas, esse projeto tem como objetivo apresentar uma ferramenta para auxiliar daltônicos a diferenciarem cores.
